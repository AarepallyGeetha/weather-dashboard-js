# Weather Dashboard

## Overview

Weather Dashboard is a responsive web application that provides real-time weather information for any city worldwide. The project demonstrates the use of asynchronous JavaScript, RESTful APIs, Fetch API, async/await, JSON data handling, and dynamic DOM manipulation.

## Features

* Search weather by city name
* Fetch real-time weather data from a public REST API
* Display temperature, humidity, wind speed, and weather conditions
* Asynchronous programming with async/await
* Dynamic UI updates without page reloads
* Comprehensive error handling for invalid cities and network issues
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* REST API
* JSON

## How to Run

1. Clone or download the repository.
2. Obtain an API key from OpenWeatherMap (if required).
3. Add the API key to the JavaScript file.
4. Open index.html in your browser.
5. Search for a city and view live weather information.
